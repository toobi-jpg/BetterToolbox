BetterToolBox

BetterToolBox acts as a improved replacement toolbox for Krita.

Start

Enable the plugin in Settings > Configure Krita > Python Plugin Manager and restart Krita. Enable the docker in Settings -> Dockers -> BetterToolBox and Disable the default Krita Toolbox.

Use

Right click a tool to open the subtools menu. Left click, hold and drag any tool to move the toolbox around when in floating mode.
